Sobre

Trabalho desenvolvido para a disciplina de Desenvolvimento de Interfaces Web do 1° período do Curso de Sistemas de Informação. Trata-se de um Portal de Games que foi construído utilizando as principais tecnologias trabalhadas durante a disciplia, dentre elas: HTML, CSS, JavaScript, Bootstrap, JSON, API, dentre outras.
